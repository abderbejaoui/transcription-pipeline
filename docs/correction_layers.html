Phonetic & Semantic Correction

Two cooperating layers. The phonetic layer casts a wide net (maximize recall); the semantic layer disposes of weak/ill-fitting candidates (restore precision).
Phonetic Layer deterministic · recall-oriented flag.py · drug_normalize.py · correction.py
1
Normalization

unlocks cross-script recall
→

Skeleton Encoding
Consonant skeleton + phonetic-class fold (p→b, v→f, c/g/q→k); CEQ classes

3
Similarity Scoring
Norm. Levenshtein; weighted Editex + Jaro-Winkler; metaphone / rapidfuzz

4
Candidate Generation
Sliding window 1–3 (3-gram→2-gram→1) to merge split drug names

5
Precision Guards
Filler blocklist, length-ratio, LCS≥3, prefix/suffix gates, hijack guard

→
6
Rank & Rescue
drug>disease tiebreaker; phonetic-alias table ("if all gone"→efferalgan)
recovers known mishearings
Phonetic layer outputs flagged spans + ranked candidates (loose thresholds 0.45–0.55) → handed to the semantic layer
